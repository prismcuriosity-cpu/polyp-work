# polyp-work
## Summary

Full implementation of the ConformalDepth research pipeline — conformal prediction applied to monocular depth estimation in colonoscopy, with guaranteed metric polyp diameter intervals.

### Model (`conformal_depth/models/`)
- `ConformalDepthModel`: Depth-Anything ViT-B/L backbone + **DV-LoRA** adapter (Q/V injections, rank-16, ~0.5% trainable params) + three-quantile DPT head (τ = α/2, 0.5, 1−α/2)
- `DVLoRAAdapter`: wraps any HuggingFace ViT via PEFT `LoraConfig`; supports merge/unmerge for fast inference
- Baselines: `DepthAnythingZeroShot`, `MonoViTWrapper`, `MCDropoutWrapper`, `DeepEnsemble`

### Conformal prediction (`conformal_depth/conformal/`)
- **Nonconformity scores**: `QuantileResidualScore`, `ScaleAlignedScore` (affine-aligned residual), `MaskRestrictedScore` (polyp-region only)
- **Calibration**: standard split-conformal quantile + **RCPS** (Angelopoulos ICML 2022) via Bentkus concentration inequality, `ConformalCalibrationResult` (serializable)
- **Video-exchangeability correction**: TV-distance proxy via domain classifier on DINOv2 CLS tokens; Wasserstein-1 slack; `ShiftAwareCoverageAdjuster` inflates threshold by ε_TV · range / √n
- **Back-projection**: `polyp_diameter_interval` uses interval arithmetic through the pinhole model — for the same pixel mask, D_lo uses depth_lo and D_hi uses depth_hi — giving a guaranteed `[D_lo, D_hi]` in mm; Paris classification (diminutive/small/large) applied to midpoint

### Training (`conformal_depth/training/`)
- **Losses**: `PinballLoss` (all 3 heads), `SILogLoss` (median only, scale-invariant), `IlluminationDeclineLoss` (inverse-square law self-supervision: log(I_j/I_i) + 2·log(d_j/d_i) = 0), `WidthRegularizationLoss`, `MonotonicityLoss` (crossing penalty)
- `ConformalDepthTrainer`: AdamW + OneCycleLR + bfloat16 AMP, periodic conformal calibration during training, TensorBoard + W&B logging

### Evaluation (`conformal_depth/evaluation/`)
- Depth: AbsRel, SqRel, RMSE, RMSElog, SILog, δ<1.25/1.25²/1.25³, Spearman ρ
- Conformal: empirical coverage vs nominal, CCDR (coverage by depth bin), width–coverage calibration curve
- Clinical: 3-group Paris accuracy, CCC, MAE/RMSE in mm, Bland–Altman, confusion matrix

### Scripts
